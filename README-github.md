# ReviewAPI-AV3
Plataforma full-stack de avaliação de mídias (filmes, séries, livros e jogos). O usuário se cadastra, importa catálogos de APIs externas para o banco local e publica avaliações com nota de 1 a 5 estrelas. Ao publicar, recebe um e-mail de confirmação com sugestões de títulos similares geradas por IA.
